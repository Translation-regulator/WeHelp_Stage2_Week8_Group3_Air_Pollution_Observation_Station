# 專案網站
## 主題： 空氣品質探測站
### https://www.airpullution.site/

### 網站功能
全台灣空氣污染指標

1. 全台即時空污詳細資訊 --> 每小時更新一次
2. 各地區探測站結合列表形式
3. 各地區一週前空氣品質歷史資料(圖表)
4. 推播當前空汙資料至 discord #bot 頻道

### 技術主題
1. SVG Graph and Animation
2. Chart.js
3. 串接 API，來源: 環境部環境資料開放平台開放資料
4. 串接 Discord Webhook
